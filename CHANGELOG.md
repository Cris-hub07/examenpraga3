# Changelog

## v1.3 - Validaciones y pruebas

- Se agregaron validaciones en la capa de logica de negocio.
- Se realizaron pruebas de compilacion y ejecucion basica por consola.

## v1.2 - Logica de negocio

- Se implementaron las clases `UsuarioService` y `AccesoService`.
- Se agregaron reglas para registro de usuarios, entradas y salidas.
- Se incorporaron funciones de reporte para historial y tiempo total en laboratorio.

## v1.1 - Acceso a datos

- Se implementaron `UsuarioDAO` y `AccesoDAO`.
- Se agrego persistencia en archivos `.txt`.
- Se incorporo lectura y escritura usando `BufferedReader` y `BufferedWriter`.

## v1.0 - Creacion de entidades

- Se crearon las entidades `Usuario` y `Acceso`.
- Se agregaron atributos, constructores, getters, setters y `toString()`.
